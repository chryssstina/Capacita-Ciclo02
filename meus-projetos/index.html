Cat Photo App

Este primeiro projeto foi feito usando apenas HTML, com a ideia de apresentar os conceitos iniciais da linguagem de marcação. O foco é mostrar as principais tags e explicar por que a semântica é importante para organizar conteúdo. E pra deixar tudo mais divertido, o exemplo prático traz alguns gatinhos fofinhos — você vai conhecer um pouco sobre eles e descobrir do que gostam, seja na hora de comer ou de brincar.

ONG Latidos & Ronrons

Este segundo projeto, feito em parceria com outro facilitador, apresenta a ONG fictícia Latidos & Ronrons. A proposta é criar um sistema simples e interativo para facilitar a adoção de animais, incentivando a adoção responsável. Também marca a introdução do CSS, deixando o visual mais organizado e com a cara da ONG.Dê uma olhada no site e conheça os animaizinhos disponíveis para adoção!

Lista de Compras

Este terceiro projeto tem como tema uma Lista de Compras, e marca a introdução ao JavaScript, com foco na manipulação do DOM. A proposta é simples, mas super útil: uma lista editável onde é possível adicionar, remover, marcar itens como comprados e até reordenar tudo usando drag-and-drop. É uma forma prática e interativa de começar a explorar o poder do JavaScript direto no navegador!
